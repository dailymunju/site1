# 웹프로젝트 테스트
## 깃 설명문서 포트폴리오 설명 정리 

처음 연결
git config --global user.name “깃이름“
git config --global user.email “가입했을때 이메일“
ctrl + j -> 터미널 
git bash
git init
git remote add origin ( 생성한 repository URL )
git remote –v (원격저장소 연결됬는지 확인)
git add . 
git status
git commit –m ‘프로젝트명’
git push origin master
